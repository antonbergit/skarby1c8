Документ "Платежное поручение"

Документ предназначен для формирования печатной формы банковских платежных поручений.

 

С помощью документа можно оформить следующие виды платежных поручений:

  • "Оплата",
  • "Перечисление в бюджет".

Операция "Оплата" предназначена для формирования печатной формы платежного поручения на оплату контрагенту.

Операция "Перечисление в бюджет"  предназначена для формирования печатной формы платежного поручения на перечисление в бюджет.

 

Для формирования печатной формы нужно для любого вида операции заполнить следующие реквизиты:

  • Организация, от которой выписывается платежное поручение. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Вид операции. Указывается вид операции документа. Предлагается выбор из списка: оплата или перечисление в бюджет.
  • Банковский счет организации, для которого оформляется платежное поручение.
  • Получатель платежа. Выбирается из справочника "Контрагенты".
  • Счет получателя платежа. Выбирается из справочника "Банковские счета".
  • Очередность платежа.
  • Оплачено - признак оплаты платежного поручения.
  • Вид платежа. Предлагается выбор из списка: электронно, почтой, телеграфом, срочно.
  • ИНН/ЕДРПОУ плательщик. Группа реквизитов:
    • ИНН. Идентификационный номер налогоплательщика плательщика.
    • ЕДРПОУ. Код ЕДРПОУ плательщика.
    • Плательщик. Отображается в соответствующем поле печатной формы документа.
  • ИНН/ЕДРПОУ получатель. Группа реквизитов:
    • ИНН. Идентификационный номер налогоплательщика получателя платежа.
    • ЕДРПОУ. Код ЕДРПОУ получателя платежа.
    • Получатель. Отображается в соответствующем поле печатной формы документа.
  • Сумма. Указывается сумма платежа.
  • Ставка НДС. Выбирается из справочника "Ставки НДС".
  • Сумма НДС. Указывается сумма НДС.
  • Назначение платежа. Текст назначения платежа. Отображается в соответствующем поле печатной формы документа.
  • Комментарий. Указывается комментарий к документу.
 
Для вида операции "Перечисление в бюджет" становится доступной для заполнения закладка "Перечисление в бюджет", на которой нужно заполнить следующие реквизиты:
  • Вид перечисления в бюджет. Указывается вид перечисления в бюджет: "Налоговый платеж", "Таможенный платеж", "Иной платеж".
  • Статус составителя. Указывается статус составителя платежного документа на перечисление налогов, сборов и иных обязательных платежей.
  • Основание платежа. Указывается условное обозначение основания налогового платежа.
  • Налоговый период. Указывается условное обозначение периода, за который перечисляется налог.
  • Номер документа. Указывается номер документа - основания перечисления  налога, сбора или иного платежа.
  • Дата документа. Указывается дата документа - основания перечисления  налога, сбора или иного платежа.
  • Тип платежа. Указывается условное обозначение типа налогового платежа.

 

Документ "Платежное поручение" может быть введен на основании документов:

  • "Заказ поставщику";
  • "Счет на оплату поставщика";
  • "Заявка на расход денег".
