Справочник "Классификатор единиц измерения"

В справочник вводится список используемых компанией единиц измерения (штуки, контейнеры, ящики, коробки и т.д.).

Справочник имеет одноуровневую структуру. Редактирование и ввод новых элементов производятся непосредственно в форме списка справочника. При заполнении справочника можно воспользоваться Классификатором единиц измерения (КЕИ).
Для добавления данных из Классификатора необходимо нажать кнопку "Подоборать из классификатора".

Откроется таблица Классификатора единиц измерения.
Непосредственно из таблицы можно добавить новую единицу измерения в справочник "Классификатор единиц измерения".

Поиск необходимой единицы измерения в Классификаторе единиц измерения можно осуществить с помощью комбинации клавиш Ctrl+F.
