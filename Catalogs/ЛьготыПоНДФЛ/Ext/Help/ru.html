Справочник «Льготы по НДФЛ»

Справочник "Льготы по НДФЛ" вызывается из главного  меню "Зарплата".

Справочник сформирован в соответствии с требованиями регламентированной формы 1ДФ и другими регламентированными формами отчетности.

Значения, принимаемые некоторыми реквизитами для расчета НДФЛ

Код соответствует номеру статьи, которая описывает эту льготу в Законе.

"Код Отчетности" может принимать значения "01", "02", "03"

"Размер" - указывает весовую долю применения льготы согласно с законом (1; 1,5; 2 для НДФЛ п.6. Закон №289-IV от 22.05.2003). Значения этого реквизита учитывается при расчете НДФЛ и используется при построении отчетности
