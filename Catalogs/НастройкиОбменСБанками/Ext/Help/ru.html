Настройки обмена с банками

Справочник используется для указания параметров обмена электронными документами с банками.

Создается настройка прямого обмена с банком, позволяющая отправлять платежные документы в банк и получать выписки из банка непосредственно из программ системы 1С:Предприятия. Это дает возможность отслеживать все движения денежных средств по расчетному счету организации.
